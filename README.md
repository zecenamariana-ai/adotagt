# AdoptaGT

Este repositorio corresponde al proyecto AdoptaGT, una propuesta de plataforma web para facilitar la adopción responsable de animales en Guatemala.

El proyecto incluye:

- Un prototipo visual sencillo en `frontend/index.html`, donde se muestran algunas mascotas de ejemplo (Tutis, Pollito Dexter, Cocoa, Sergia, Tomsito y Jorgito).
- Un archivo con la definición de los endpoints principales de la API en `backend/api_ejemplo.txt`.

Este repositorio acompaña el documento en PDF donde se describe la arquitectura tecnológica, el diseño técnico, el modelo de base de datos, la API y la estimación de horas y costos del proyecto.
